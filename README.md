# VueAuth — Практическая работа №2 (Figma Redesign)

SPA-приложение на Vue 3, реализованное в точном соответствии с предоставленным дизайн-макетом из Figma.

##  Дизайн и интерфейс

Приложение переверстано с учетом визуальных референсов:
- **Цветовая схема**: Белый контентный блок на светло-сером фоне, акцентный коралловый цвет (`#ff6464`) для кнопок и активных ссылок.
- **Типографика**: Использование системных шрифтов с акцентом на заголовок 44px и основной текст 16px.
- **Компоненты**: Rounded-карточки с мягкими тенями для списка пользователей и формы входа.
- **Layout**: Фиксированный контейнер 1100px с центрированным содержимым.

##  Функционал

1.  **Home (`/home`)**:
    - Заголовок "Home page".
    - Иллюстрация инструментов (SVG).
    - Кнопка "Authorization".
2.  **Users (`/users`)**:
    - Список из 5 пользователей (DummyJSON).
    - Отображение ФИО и Email в карточках.
3.  **Login (`/login`)**:
    - Центрированная форма.
    - Обработка ошибки "Invalid credentials" красным цветом в заголовке формы.
    - Перенаправление в профиль при успехе.
4.  **Profile (`/profile`)**:
    - Приватный маршрут (Auth Guard).
    - Вывод данных: Username, Name, Lastname, Gender, Email.
    - Отображение аватара пользователя.
    - Вывод текста ошибки от сервера в заголовке при невалидном токене.
